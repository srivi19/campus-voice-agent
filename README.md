# CampusVoice 🎓

**AI-powered student sentiment agent for universities — built with Gemini + Elasticsearch**

CampusVoice aggregates real student reviews from Rate My Professors, indexes them in Elasticsearch with semantic embeddings, and lets administrators, counselors, and prospective students ask plain-English questions to surface what's really happening on campus.

---

## What It Does

Instead of manually reading hundreds of reviews, you can ask:

> *"What are students at UTK complaining about most?"*
> *"How does Vanderbilt compare to UTK for engineering students?"*
> *"What do students say about intro CS courses?"*

The agent searches semantically across thousands of real student reviews and synthesizes a structured, evidence-backed answer with real quotes.

## Setup

### 1. Prerequisites

- Python 3.8+
- Google Cloud account with Agent Platform (Vertex AI) enabled
- Elastic Cloud account (free trial works)
- Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey)

### 2. Install dependencies

```bash
pip install google-genai elasticsearch praw beautifulsoup4 requests python-dotenv
```

### 3. Configure environment

Create a `.env` file in the project root:

```
GEMINI_API_KEY=your_gemini_api_key
ELASTICSEARCH_URL=your_elasticsearch_endpoint
ELASTICSEARCH_API_KEY=your_elasticsearch_api_key
```

### 4. Run the pipeline

```bash
# Step 1: Collect reviews from Rate My Professors
python collect_rmp.py

# Step 2: Create the Elasticsearch index
python setup_index.py

# Step 3: Embed and index the reviews
python ingest.py

# Step 4: Chat with your agent
python chat.py
```
